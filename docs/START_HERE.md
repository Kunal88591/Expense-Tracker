## ⚠️ URGENT: Credential Security

**You shared your MongoDB password in chat.**

**You must rotate it immediately:**

1. Go to MongoDB Atlas
2. Security → Database Access
3. Delete old user
4. Create new user with new password
5. Update your local `.env`

See: [SECURITY_GUIDE.md](SECURITY_GUIDE.md)

**Takes 5 minutes. Do this first.**
